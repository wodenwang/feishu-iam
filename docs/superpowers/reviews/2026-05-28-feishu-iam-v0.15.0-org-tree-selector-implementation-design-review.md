## 重要

- 已修复实现中残留的旧飞书同步内部 IA：`SystemSettingsView` 不再保留隐藏的“本地部门树 / 本地用户列表”旧结构，首屏只保留健康摘要、`OrgBrowser` 组织用户浏览、字段诊断、同步历史和高级/危险全量同步区。
- `OrgBrowser` 使用下钻模型、搜索、加载更多、空态和错误卡片；错误卡片明确说明只影响当前浏览区域，不清空已选草稿或其他排障区域。
- `OrgUserSelector` 桌面使用待选浏览器 + 已选/摘要右栏；窄屏使用 `待选 / 已选 / 摘要` Tabs，摘要区有保存中、保存失败和草稿保留说明。
- 选择语义在 UI 中明确：部门主体不等于自动绑定全部用户，用户可单独选择，重复覆盖不会重复保存。
- #24 现有应用详情角色管理表格保持 132px 稳定操作列，查看/编辑/启停具备 `aria-label` 和 `title`，停用/启用使用确认弹框。
