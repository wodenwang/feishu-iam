# v0.1.11 应用接入配置设计复核

范围：应用详情、接入配置页、权限注册结果、README 入口文档。

## 结论

通过。实现沿用既有 Ant Design Admin Console 信息架构，没有重新设计后台框架。v0.1.11 只补齐 HTTP runtime 下的应用接入配置闭环：列表进入详情、详情进入接入配置、配置页展示脱敏 runtime 信息、权限注册结果和接入检查状态。

## 复核要点

- 应用列表在 HTTP mode 下显示 `查看` 和 `接入配置`，避免出现 mock-only 禁用操作。
- 应用详情 HTTP mode 下保留查看能力，隐藏停用和轮换 secret 等未实现危险操作。
- 接入配置页不展示 App Secret 明文，只展示 `<copy-from-admin-console>` 占位。
- Agent Prompt 继续强调不得请求、记录或提交 App Secret、token、导出用户列表。
- 权限注册结果来自 runtime API，不再依赖 mock-only 数据。
- README 改为公开项目入口，优先说明项目价值、快速启动和快速部署，不再承载机器私有部署细节。
