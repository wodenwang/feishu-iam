## 关键约束

- 本版本只处理 GitHub issue `#2`，不新增 DDL，不扩大 OAuth/OIDC 协议面，不改变第三方应用 `redirect_uri` 精确匹配规则。
- `FEISHU_OAUTH_REDIRECT_URI` 推荐路径为 `/oauth/feishu/callback`，旧 `/api/auth/feishu/callback` 只作为兼容入口。
- 不在文档、日志、测试或归档中记录 secret、token、cookie、authorization code、飞书 `app_secret`、数据库密码或 raw payload。

## 主要修改

- 新增兼容控制器 `apps/api/src/oauth/legacy-oauth-callback.controller.ts`，让 `GET /api/auth/feishu/callback` 复用现有 OAuth 回调处理。
- 更新 `apps/api/src/oauth/oauth.service.ts`，集中校验 Feishu 内部 OAuth 回调路径，并在 code exchange 时使用同一 `redirect_uri`。
- 更新 `apps/api/src/oauth/oauth-error.filter.ts`，让兼容旧路径也渲染统一 HTML 错误页。
- 补齐 OAuth service、filter 和 e2e 测试。
- 同步 `package.json`、`apps/api/package.json`、`apps/admin-web/package.json`、`deploy/install.sh`、`deploy/server.env.example`、`README.md`、`CHANGELOG.md` 和 `AGENTS.md` 到 `v1.0.1`。
