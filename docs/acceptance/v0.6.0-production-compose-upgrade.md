# Feishu IAM v0.6.0 生产化部署验收清单

本文档用于验收 Feishu IAM `v0.6.0` 在 `http://feishu-iam.example.com` 的单机 Docker Compose 部署、停机静态升级、首个平台管理员初始化和破窗入口退出。

验收过程中不要把真实密钥、token、cookie、密码、authorization code、access token 或 client secret 写入仓库、文档、截图、聊天消息、工单或会话归档。临时明文 client secret 只允许保存在当前终端会话或受控密钥介质中，用完即清理。

## Compose 验收

- [ ] `docker compose config --quiet` 通过。
- [ ] `docker compose ps` 显示 `db` 和 `web` 正常运行。
- [ ] `docker ps` 中容器名带 `feishu-iam` 前缀，例如 `feishu-iam-db-1` 和 `feishu-iam-web-1`。
- [ ] 宿主机未暴露 PostgreSQL `5432` 端口。
- [ ] Web 端口按 `.env` 映射到宿主机 `8000`。
- [ ] `http://192.168.2.112:8000/health` 返回服务健康。
- [ ] `http://192.168.2.112:8000/ready` 返回 `ready`。
- [ ] `http://192.168.2.112:8000/version` 返回 `0.6.0`。
- [ ] `http://feishu-iam.example.com/version` 返回 `0.6.0`。
- [ ] 浏览器访问 `http://feishu-iam.example.com` 可以打开管理后台。

## 目录和挂载验收

- [ ] 服务器部署目录为 `~/feishu-iam`。
- [ ] 根目录只保留 `docker-compose.yaml`、`.env`、`upgrade.sh` 和必要运行目录。
- [ ] `config/` 已映射到容器 `/app/config`。
- [ ] `logs/` 已映射到容器 `/app/logs`。
- [ ] `data/postgres/` 保存 PostgreSQL 数据。
- [ ] `backups/` 保存升级脚本生成的数据库备份。

## 升级验收

- [ ] `./upgrade.sh` 可以拉取目标 `web` 镜像。
- [ ] `./upgrade.sh` 会先停止 `web`。
- [ ] `./upgrade.sh` 在 `backups/` 下生成数据库备份。
- [ ] `./upgrade.sh` 通过 `web` 镜像内的 `/app/deploy/apply-migrations-in-container.sh` 执行 DDL。
- [ ] `schema_versions` 存在 `0.6.0`。
- [ ] DDL 重跑不会重复应用已存在版本。
- [ ] 升级完成后 `/ready` 返回 `ready`。
- [ ] 升级完成后 `/version` 精确返回 `0.6.0`。

## 管理员验收

- [ ] `INITIAL_PLATFORM_ADMIN_FEISHU_USER_ID` 指向“王文哲”的真实飞书 `user_id`。
- [ ] “王文哲”可通过飞书登录管理后台。
- [ ] `/api/v1/admin/me` 返回角色包含 `platform_admin`。
- [ ] “王文哲”可以创建应用管理员。
- [ ] 应用管理员可以按授权范围管理应用。
- [ ] 生产环境 `/admin/auth/bootstrap` 不可访问。
- [ ] 未登录管理端只展示飞书登录入口。

## 飞书与 SSO 回归验收

- [ ] 飞书开放平台已登记 `http://feishu-iam.example.com/admin/auth/feishu/callback`。
- [ ] 飞书开放平台已登记 `http://feishu-iam.example.com/oauth/feishu/callback`。
- [ ] 飞书通讯录同步仍能成功完成。
- [ ] 测试应用可以通过 `/oauth/authorize` 发起登录。
- [ ] 授权码可以通过 `/oauth/token` 换取 Feishu IAM access token。
- [ ] `/oauth/userinfo` 返回当前登录用户身份。
- [ ] `/api/v1/apps/{app_key}/me/permissions` 返回当前用户在应用下的权限组和权限点。
- [ ] `/oauth/revoke` 后 token 不再可用。

## 安全验收

- [ ] `.env` 权限为 `600`。
- [ ] 文档和会话归档不包含真实密钥、token、cookie、密码或 client secret。
- [ ] 审计日志不记录飞书 `app_secret`、平台 token 或 client secret。
- [ ] PostgreSQL 只在 Compose 网络内访问，宿主机没有 `5432` 端口映射。
- [ ] 生产路径不再配置 `BOOTSTRAP_SUPER_ADMIN_*`。
