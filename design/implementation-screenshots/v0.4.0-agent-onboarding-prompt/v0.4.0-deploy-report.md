## 部署动作

1. 使用 `git archive --format=tar HEAD` 将仓库受控文件同步到远端部署目录。
2. 保留远端 `.env` 与 `data/postgres`，并创建 `.env.backup-v0.4.0-20260528164714`。
3. 将远端 `.env` 中的 `FEISHU_IAM_IMAGE_TAG` 更新为 `v0.4.0`。
4. 在远端执行 `docker compose up -d --build --remove-orphans` 完成镜像构建、容器重建和启动。

## 边界确认

- 本版本只完善第三方应用 Agent 接入初始化提示词、一次性 secret 提示词复制和相关审计语义。
