# Codex 会话归档：v1.0.0 Riversoft Logo 修正

## 会话目标

修正 Feishu IAM v1.0.0 Riversoft 正式版第一切片中后台 Sidebar 使用了非 Riversoft logo 的问题，并重新执行设计审查。

## 用户关键要求

- 当前截图中的 logo 是其他公司的，并非 Riversoft。
- 从 Riversoft logo 中截取内容替换。
- 修改完成后重新使用 `gstack /design-review` 做审查。

## 关键约束

- 只修正前端品牌视觉资产和品牌区文案。
- 不改变 UX 流程、信息架构、路由、表单流程、数据契约、CRUD 行为、权限逻辑或业务逻辑。
- 不提交真实 secret、token、cookie、飞书 app_secret、数据库密码或生产导出数据。

## 重要设计决策

- 使用 `DESIGN.md` 已记录的 Riversoft 官方 logo 素材 `https://www.riversoft.com.cn/images/logo-2.png`。
- 由于当前 Sidebar 品牌位是 40px 方形图标位，横向 logo 全量缩放会不可读，因此截取 logo 左侧完整圆形图形内容生成 72px 方形 PNG。
- 首次按 47px 方形裁剪会切掉右侧蓝色边缘；修正后使用 ImageMagick 按 `50x47+0+0` 裁剪，再居中放入 72×72 透明画布，保留完整圆形图形。
- 同步替换后台品牌图和浏览器 favicon，避免同一版本中出现两套品牌标识。
- 将品牌区辅助文案从旧公司名称调整为 `Riversoft 内部身份与权限控制台`，避免 logo 修正后仍保留品牌误导。
