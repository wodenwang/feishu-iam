## 重要设计决策和原因

- `v1.0.1` 只处理 GitHub issue `#2`：内部 Feishu OAuth 回调地址配置与实际路由不一致导致第三方 SSO 登录 404。
- 补丁范围锁定为旧回调路径兼容、内部回调配置校验、部署样例和真实 SSO 验收。
- 不扩大 OAuth/OIDC 协议面，不新增 DDL，不改变第三方应用 `redirect_uri` 精确匹配规则。
- 推荐生产配置继续使用 `/oauth/feishu/callback`；`/api/auth/feishu/callback` 仅作为兼容入口，避免旧配置直接落到框架 404。
