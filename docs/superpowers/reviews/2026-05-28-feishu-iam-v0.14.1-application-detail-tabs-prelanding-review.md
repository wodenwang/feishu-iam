# Feishu IAM v0.14.1 应用详情 Tab 化落地前审查

日期：2026-05-28
审查方式：本机手工代码审查。当前环境没有可用的 `gstack` 命令，因此用等价的 diff 阅读、测试、构建和 Playwright 浏览器检查替代自动 review skill。

## 结论

未发现阻塞发布的问题。当前剩余工作是版本元数据、README/CHANGELOG、提交、推送、MR 合并、tag、GitLab release、镜像发布和 `192.168.2.112` 停机升级验证。

## 重点检查

- 路由边界：只在应用详情页新增 `tab` 查询参数状态，没有新增后端 API、DDL、Prisma schema 或 SSO 协议面。
- 兼容边界：旧 `sheet=app:*` 抽屉入口仍保留，并通过本地 Tab state 工作。
- URL 边界：非法 `tab` 回退到 `详细资料`；切换到 `详细资料` 时删除多余 `tab` 参数；其他 Tab 保留 `from` 等已有查询参数。
- 权限边界：权限管理未恢复角色元数据新增、编辑或启停；角色元数据仍归应用管理。
- 安全边界：未新增明文 secret、token、cookie 或真实密码；安全版接入提示词仍不包含敏感凭证。
- UI 边界：桌面和移动视口未发现横向溢出、元素遮挡或异常空白。
- 测试边界：新增 App 路由测试和应用管理测试覆盖 Tab 深链、非法 Tab 回退、切换保留 `from`、各 Tab 下的关键交互。

## 风险

- 本次是前端信息架构补丁，不改变数据库和后端契约；线上风险主要来自详情页入口和 Tab URL 状态。现有测试与 Playwright 自检已经覆盖该风险。
- 本机没有真实飞书管理员 session，浏览器自检使用 API mock；生产最终验收需要在 `192.168.2.112` 部署后确认 `/ready`、`/version` 和未登录后台入口的预期认证行为。

## 结论

可以进入 Git closeout 和 ship 阶段。
