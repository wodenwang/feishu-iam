日期：2026-05-24
范围：`/login` 登录入口、callback 失败状态、初始化入口。

## 结论

通过。当前切片没有新增页面结构，只接线既有登录按钮，并新增 `authFailed` 状态文案。

## 检查

- 1440px 登录页可见 `使用飞书登录` 主按钮。
- HTTP runtime 下本地开发仍显示 `使用本地 mock 飞书登录`，不影响生产真实入口。
- 登录页没有 username/password 字段。
- callback 失败状态复用现有 Ant Design `Result` 结构。
- favicon 404 已通过 `index.html` data favicon 消除，浏览器 console 重新检查为 0 error / 0 warning。

## 发现与修复

1. 发现：首次浏览器检查出现 `/favicon.ico` 404。
