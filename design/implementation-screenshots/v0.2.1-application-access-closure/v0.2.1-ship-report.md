# v0.2.1 应用接入闭环补缺 Ship 收口记录

日期：2026-05-28

## 范围

当前 ship 收口覆盖 `v0.2.1` 应用接入闭环补缺：

- 角色授权支持绑定权限组，并在用户权限查询中展开 active 权限组下的 active 权限点。
- 应用详情接入配置支持一键复制第三方应用接入提示词。
- 未登录跳转场景显示明确的飞书重新登录提示。
- 版本元数据更新到 `0.2.1`。

明确不包含：

- username/password、本地 root 或独立账号体系。
- 完整 SDK、CLI、OIDC/JWKS/refresh token 或多租户 IdP。
- `/directory` 独立组织编辑能力。
