## 部署方式

远端目录不是 Git checkout。本次使用 `git archive HEAD` 流式同步已提交文件到 `/home/bpmt/feishu-iam`，保留远端 `.env` 与 `data/`，然后执行：

```bash
FEISHU_IAM_IMAGE_TAG=v0.1.10 docker compose build feishu-iam
FEISHU_IAM_IMAGE_TAG=v0.1.10 docker compose up -d --remove-orphans
```
