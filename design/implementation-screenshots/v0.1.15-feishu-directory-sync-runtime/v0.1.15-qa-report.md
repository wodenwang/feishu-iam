## Findings

发现并修复：

1. HTTP mode 仍把 `/sync` 视为 disabled route，导致跳回 `/applications`。
   - 修复：`src/app/App.tsx` 加入 `/sync`，`src/layouts/AdminLayout.tsx` 对平台管理员显示同步菜单。
2. 前端同步摘要的 `用户数量` 使用后端 `success_count`，后端原先把部门和用户都计入。
   - 修复：`success_count` 改为用户数量；部门变化继续由 diff summary 显示。
3. Playwright 三视口并发时 toast 断言不稳定。
   - 修复：回归测试以成功表格行和同步后的目录用户作为稳定完成证据。
