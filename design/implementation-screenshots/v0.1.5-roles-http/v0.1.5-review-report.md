## Finding

1. `src/pages/Roles/index.tsx` 编辑态允许修改角色所属应用和角色编码，但 HTTP `PATCH /api/roles/:id` 不支持修改这两个字段。
   - 风险：用户保存后看到成功提示，但字段变更被忽略，造成授权归属误判。
   - 修复：编辑态禁用 `所属应用` 和 `角色编码`，仅新建态可编辑。
   - 复验：`npm run test -- src/pages/Roles/index.test.tsx` 通过；`npm run build` 通过。
2. `src/pages/Roles/index.tsx` 导出测试 helper `formatRoleError`，触发 Vite Fast Refresh 兼容性警告。
   - 风险：本地开发热更新失效或需要整页刷新，降低前端调试效率。
   - 修复：新增 `src/pages/Roles/errors.ts` 承载 helper，组件文件只导出组件。
   - 复验：`npm run test -- src/pages/Roles/index.test.tsx` 通过；`npm run build` 通过。
