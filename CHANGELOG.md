# 更新日志

## 0.1.0 - 2026-05-23

- 建立 React、TypeScript、Vite、Ant Design Admin Console 前端骨架。
- 增加飞书 IAM v0.1 产品规格、Pencil 原型、实现截图和 QA 记录。
- 增加应用管理、应用详情、接入向导、角色授权、组织用户、同步中心、审计日志、登录、初始化、403 和全局错误页面。

- 增加 mock IAM service、TanStack Query hooks、组件测试和 Playwright 三视口 E2E。
- 保持飞书唯一身份源边界，不新增本地 username / password 登录或本地超级管理员体系。
