## 发现与处理

1. HTTP runtime 左侧菜单缺少 `/roles` 和 v0.1.4 已开放的 `/directory`。
   - 处理：更新 `src/layouts/AdminLayout.tsx` 的 HTTP 菜单白名单。
2. 编辑角色时 UI 允许修改 `所属应用` 和 `角色编码`，但 HTTP 后端不支持变更这些字段。
   - 处理：编辑态将 `所属应用` 和 `角色编码` 设为 disabled。
