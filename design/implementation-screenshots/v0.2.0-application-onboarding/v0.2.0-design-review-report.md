# v0.2.0 应用接入生产化 Design Review

日期：2026-05-27

## 结论

通过。实现版 Admin Console 与已确认 Pencil 原型方向一致，保留 Ant Design 中后台密度和表格优先结构，覆盖概览、接入配置、redirect URI Drawer、停用确认、secret 轮换确认、一次性结果、应用管理员、最后管理员保护、审计筛选和 768px 响应式。

## 已发现并关闭的问题

| 级别 | 问题 | 修复 | 复验 |
|---|---|---|---|
| P1 | 768px 下 redirect URI 表格因为禁用横向滚动出现竖向断字。 | 真实浏览器启用 Table `scroll.x`，jsdom 测试环境禁用滚动以避免 Ant Design 测试挂起。 | `12-config-768.png` 已复验，URI 不再断成竖排。 |
| P1 | 最后管理员保护接口返回 `code=LAST_APPLICATION_ADMIN`，页面只检查 `error.message`，没有显示指定保护文案。 | 改为同时识别 `IamHttpError.code` 和错误 message。 | `08-last-admin-protection.png` 已复验。 |
