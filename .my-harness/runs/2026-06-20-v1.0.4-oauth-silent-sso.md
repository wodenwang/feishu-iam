# v1.0.4 OAuth silent SSO run

## 目标

支持第三方应用在 Base Portal iframe 场景下安全使用 Feishu IAM silent SSO。

## 关键决策

- 使用标准 `prompt=none` 参数。
- 新增 IAM SSO browser session，不复用管理员 session。
- 应用级策略默认关闭，通过 `silent_sso_enabled` 和 `silent_sso_allowed_origins` 治理。
- `feishu-iam-sso-demo` 迁移预置生产 origin。
- Base Portal 仍只做 iframe 宿主和入口编排，不传 token、cookie、authorization code 或 secret。

## 修改范围

- OAuth authorize / callback / error filter
- Prisma schema 与 `V1_0_4` 迁移
- 应用配置模型
- Docker Compose / deploy env 默认值
- README、CHANGELOG、SSO 接入、排障、部署和验收文档
