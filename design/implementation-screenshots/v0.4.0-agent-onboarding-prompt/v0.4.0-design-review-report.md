# v0.4.0 设计复核记录

日期：2026-05-28

## 复核范围

- 应用创建成功后的一次性 secret modal。
- 应用详情页 `接入配置` 中的普通 Agent 初始化提示词卡片。
- appSecret / API secret 轮换后的一次性 secret modal。

## 结论

- 接入体验符合本版本目标：第三方系统开发 Agent 可以直接复制一段初始化提示词，并生成 `AGENTS.md` / `CLAUDE.md`、接口说明、环境变量和 SOP。
- 普通应用详情提示词与一次性 secret 提示词边界清晰：普通场景只给占位符，创建和轮换结果只暴露本次明文 secret。
- 交互沿用现有 Ant Design Card、Alert、Modal 和 Button，不引入新的设计语言，也不改变应用管理主流程。
- 页面密度和信息组织符合 Admin Console 习惯，未把 CRUD 列表改成卡片化主界面。

## 设计取舍

- 本版本没有新增 Pencil 原型，因为功能复用现有应用详情页与 secret modal 交互，只增加一个复制入口和提示词内容。
- 没有新增 SDK 下载、CLI、外部文档站或多语言模板，避免扩大 `v0.4.0` 边界。
