## 重点检查

| 维度 | 结论 |
|---|---|
| 版本边界 | 版本号已推进到 `0.2.1`，范围聚焦应用接入闭环补缺。 |
| 领域边界 | 保持飞书唯一身份源；未新增 username/password、本地 root 或独立账号体系。 |
| SQL/migration | 新增 `009_role_permission_groups.sql`，角色和权限组建立独立绑定表。 |
| 后端权限 | 角色授权保存权限组；应用权限查询会展开 active 权限组下 active 权限点。 |
| 前端权限 | 角色授权 Tree 区分权限组 key 与权限点 key，提交 `permissionGroupCodes` 与 `permissionPointKeys`。 |
| 提示词复制 | 应用详情提供一键复制 Agent 提示词，包含 SSO、redirect URI、Application API 和 HMAC 规则。 |
| Secret 安全 | 提示词仅列出环境变量名和鉴权规则，不直接泄漏真实 secret 明文。 |
| 审计 | 复制提示词动作沿用 secret copy audit 入口，记录 `kind=agent_prompt`。 |
| 测试覆盖 | 后端、前端页面、HTTP payload、migration、构建和浏览器主路径均已覆盖。 |
